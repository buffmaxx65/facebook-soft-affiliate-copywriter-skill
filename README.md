# Facebook Soft Affiliate Copywriter — OpenClaw Skill

Skill untuk membuat copywriting Facebook affiliate yang natural, edukatif, storytelling, dan tetap transparan.

## Fungsi utama

- Membuat post utama Facebook
- Membuat komentar/utas lanjutan
- Menyisipkan affiliate link secara natural
- Menulis CTA halus dan disclosure
- Menghindari klaim menyesatkan

## Install

```bash
git clone https://github.com/buffmaxx65/facebook-soft-affiliate-copywriter-skill \
  ~/.openclaw/skills/facebook-soft-affiliate-copywriter
```

Atau:

```bash
curl -fsSL https://raw.githubusercontent.com/buffmaxx65/facebook-soft-affiliate-copywriter-skill/main/install.sh | bash
```

## Cara pakai

```bash
openclaw agent --message "Buat post Facebook dan komentar lanjutan untuk produk spray mop dengan link affiliate"
```

## Output

1. Brief konten
2. Post utama Facebook
3. Komentar/utas lanjutan
4. Variasi hook
5. Catatan compliance
